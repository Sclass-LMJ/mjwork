# 부가가치세 신고서 텍스트 파싱 도구 사용방법

## 개요

OCR로 변환된 부가가치세 신고서 텍스트 파일에서 페이지 1, 2의 데이터를 자동으로 추출하여 Excel로 정리하는 VBA 매크로입니다.

## 파일 설명

- **changetexttoexcel.bas**: VBA 매크로 코드
- **'25년도 1기 확정 부가가치세 신고서 등\_text전환.txt**: 샘플 텍스트 파일

## 설치 방법

### 1. Excel 매크로 활성화

1. Excel을 실행합니다
2. `파일` > `옵션` > `보안 센터` > `보안 센터 설정` 클릭
3. `매크로 설정`에서 "모든 매크로 포함" 또는 "디지털 서명된 매크로만" 선택
4. 확인 클릭

### 2. VBA 코드 가져오기

#### 방법 A: 직접 가져오기

1. Excel에서 `Alt + F11`을 눌러 VBA 편집기 열기
2. `파일` > `파일 가져오기` 선택
3. `changetexttoexcel.bas` 파일 선택
4. 가져오기 완료

#### 방법 B: 수동으로 복사

1. Excel에서 `Alt + F11`을 눌러 VBA 편집기 열기
2. `삽입` > `모듈` 선택
3. `changetexttoexcel.bas` 파일의 내용을 모두 복사하여 붙여넣기
4. 저장 (`Ctrl + S`)

### 3. 매크로 사용 통합 문서로 저장

1. `파일` > `다른 이름으로 저장`
2. 파일 형식을 **"Excel 매크로 사용 통합 문서 (\*.xlsm)"**로 선택
3. 저장

## 사용 방법

### 1. 매크로 실행

1. Excel에서 `Alt + F8` 키를 누릅니다
2. 매크로 목록에서 **"부가가치세신고서\_파싱"** 선택
3. `실행` 버튼 클릭

### 2. 텍스트 파일 선택

- 파일 선택 대화상자가 나타나면 변환된 텍스트 파일을 선택합니다
- 예: `'25년도 1기 확정 부가가치세 신고서 등_text전환.txt`

### 3. 결과 확인

- 새 시트 "부가세신고서"가 생성되고 다음 내용이 정리됩니다:
  - **페이지 1**: 일반과세자 부가가치세 신고서
    - 사업자 정보
    - 과세표준 및 매출세액
    - 매입세액
    - 납부(환급)세액
  - **페이지 2**: 명세 항목 상세
    - 그 밖의 공제 매입 세액 명세
    - 공제받지 못할 매입 세액 명세
    - 경감·공제 세액 명세
    - 가산세액 명세

## 주요 기능

### 1. 자동 페이지 구분

- "--- 페이지 1 ---", "--- 페이지 2 ---" 마커를 기준으로 자동 분리

### 2. 데이터 추출

- OCR 텍스트에서 금액, 세율, 세액 자동 추출
- 쉼표가 포함된 숫자 처리
- 음수(환급) 처리

### 3. 서식 자동 적용

- 헤더에 색상 적용
- 숫자에 천단위 구분 쉼표 자동 적용
- 열 너비 자동 조정
- 테두리 자동 적용

## 텍스트 파일 형식 요구사항

텍스트 파일은 다음 형식이어야 합니다:

```
--- 페이지 1 ---
[페이지 1 내용]

--- 페이지 2 ---
[페이지 2 내용]

--- 페이지 3 ---
...
```

## 문제 해결

### 매크로가 보이지 않아요

- VBA 편집기(`Alt + F11`)에서 모듈이 제대로 가져와졌는지 확인하세요
- 프로젝트 탐색기에 "부가가치세파싱" 모듈이 있어야 합니다

### "파일을 읽을 수 없습니다" 오류

- 파일이 UTF-8 인코딩인지 확인하세요
- 파일 경로에 한글이 포함되어 있는지 확인하세요
- 파일이 열려있지 않은지 확인하세요

### "페이지 1 내용을 찾을 수 없습니다" 오류

- 텍스트 파일에 "--- 페이지 1 ---" 마커가 있는지 확인하세요
- 파일 인코딩이 올바른지 확인하세요

### 값이 제대로 추출되지 않아요

- OCR 품질이 낮으면 숫자가 제대로 인식되지 않을 수 있습니다
- 텍스트 파일을 열어서 숫자가 올바르게 변환되었는지 확인하세요

## 개선사항 (원본 대비)

### ✅ 수정된 문제점

1. **파일 확장자**: `.module` → `.bas`로 변경
2. **실제 파싱 구현**: 하드코딩된 값 대신 텍스트에서 자동 추출
3. **페이지 구분**: `ExtractPageContent()` 함수로 페이지별 분리
4. **ByRef 매개변수**: `rowNum`을 ByRef로 전달하여 올바르게 증가
5. **숫자 처리**: `CleanNumber()` 함수로 쉼표 제거 및 변환
6. **에러 처리**: 각 단계마다 에러 처리 추가

### 🎨 추가된 기능

1. **자동 시트 생성**: "부가세신고서" 시트 자동 생성
2. **서식 적용**: 색상, 테두리, 정렬 자동 적용
3. **페이지별 구분**: 페이지 1, 2를 명확하게 구분하여 표시
4. **유연한 파일 읽기**: UTF-8 인코딩 지원
